CSS Media Queries Demonstration

Media queries allow web pages to be responsive by applying different styles based on screen size, orientation, or user preferences.

1. Basic Responsive Design (Max-Width)

This example changes the background when the viewport width is **600px or less**.

Resize the window to see the effect!

@media (max-width: 600px) {
  .example { background-color: lightblue; }
}

2. Min-Width for Larger Screens

When the viewport is **800px or wider**, the background color changes.

Resize to a larger screen to see the effect!

@media (min-width: 800px) {
  .example { background-color: lightgreen; }
}

3. Orientation Based Styling

Changes background color when the device is in **landscape mode**.

Rotate your device to see the effect!

@media (orientation: landscape) {
  .example { background-color: orange; }
}

4. Dark Mode Support

If the user prefers **dark mode**, this section will change its appearance.

Try enabling dark mode in your system settings.

@media (prefers-color-scheme: dark) {
  .example { background-color: black; color: white; }
}

5. Print-Specific Styles

This section will have a different style **when printed**.

Try printing this page to see the effect.

@media print {
  .example { background-color: white; color: black; font-style: italic; }
}
